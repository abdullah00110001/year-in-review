## 📖 Overview

Sukoon OS is an all-in-one Islamic life operating system that helps users track daily habits, monitor spiritual progress, manage goals, and build discipline — all through an Islamic lens. Available as a web app and native Android application.

---

- **Push Notifications** — Firebase Cloud Messaging
- **Local Notifications** — Alarm and reminder support
- **Deep Linking** — Supabase auth callback handling
- **Back Button** — Smart dialog/drawer dismissal
- **Status Bar** — Themed dark status bar
- **Haptic Feedback** — Native vibration support
- **Camera** — Photo capture for Rise missions
- **Share** — Native share sheet integration
- **Keyboard** — Smart keyboard handling

### Build Instructions

```bash
# 1. Install dependencies
npm install

# 2. Build the web app
npm run build

# 3. Sync with Android
npx cap sync android

# 4. Run on device/emulator
npx cap run android
```

### Production APK

```bash
# Build release APK
cd android
./gradlew assembleRelease
```

The APK will be at `android/app/build/outputs/apk/release/app-release.apk`

## 🗄️ Database Schema

The Supabase database contains **70+ tables** including:

| Category | Tables |
|----------|--------|
| **Core** | `daily_entries`, `daily_logs`, `activity_tags`, `daily_intentions` |
| **Goals** | `goals`, `habits`, `habit_entries`, `challenges`, `user_challenges` |
| **Islamic** | Integrated into `daily_entries` (salah, quran, tahajjud fields) |
| **Shield** | `discipline_profiles`, `discipline_scores`, `focus_sessions` |
| **Rise** | `rise_alarms`, `rise_alarm_logs`, `group_wake_status` |
| **Life Calendar** | `life_profile`, `life_weeks`, `life_milestones` |
| **Social** | `accountability_groups`, `accountability_group_members`, `wake_signals` |
| **Gamification** | `user_gamification`, `user_challenges`, `challenges` |
| **Admin** | `admin_roles`, `admin_feedback`, `admin_alerts`, `app_settings` |
| **Payments** | `user_subscriptions`, `payment_transactions`, `invoices`, `coupons` |
| **Knowledge** | `knowledge_items`, `future_letters`, `reflections` |
| **System** | `feature_flags`, `behavior_rules`, `governance_policies`, `device_intelligence` |

All tables use **Row Level Security (RLS)** to ensure data isolation per user.

## 🚀 Getting Started

### Prerequisites
- Node.js 18+
- npm or bun

### Development

```bash
# Clone the repository
git clone <repository-url>

# Install dependencies
npm install

# Start development server
npm run dev
```

The app will be available at `http://localhost:8080`

### Environment Variables

The following are auto-configured via Supabase integration:
- `VITE_SUPABASE_URL`
- `VITE_SUPABASE_PUBLISHABLE_KEY`
- `VITE_SUPABASE_PROJECT_ID`
